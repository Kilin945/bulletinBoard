# Claude 專案資訊

## 專案概述
此專案是一個佈告欄系統 (Bulletin Board)。

## 開發環境
- 平台: macOS (Darwin 24.5.0)
- Git 倉庫: 是
- 主分支: main

## 專案結構
```
/
├── .vscode/          # VS Code 設定檔
├── pom.xml          # Maven 專案設定檔
├── src/             # 原始碼目錄
└── target/          # 編譯輸出目錄
```

## 開發指令
請在此處添加常用的開發指令，例如：
- 編譯: `mvn compile`
- 測試: `mvn test`
- 執行: `mvn exec:java`
- 清理: `mvn clean`
